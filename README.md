Fungsi: Membantu guru memasukkan nilai ujian dan menghitung nilai rata-rata
Cara Kerja:
Menginput jumlah siswa sebagai ukuran data
Menggunakan loop untuk memasukkan nilai setiap siswa dan menyimpan ke array
Menjumlahkan semua nilai dan kemudian menghitung rata-rata dengan membagi total nilai terhadap jumlah siswa
Menampilkan semua nilai dan hasil rata-rata
